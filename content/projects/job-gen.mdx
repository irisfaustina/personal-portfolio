# Vibecoder Job Description Generator

A modern, AI-powered web application that automatically generates professional job descriptions from GitHub repositories. Perfect for freelancers and agencies looking to quickly create compelling project descriptions for platforms like Fiverr and Upwork.

## 🚀 What it does

The Vibecoder Job Description Generator analyzes any public GitHub repository and creates comprehensive, professional job descriptions by:

- **Smart Repository Analysis**: Automatically detects frontend and backend technologies, programming languages, and project structure
- **AI-Powered Content Generation**: Uses advanced AI to understand project scope and generate tailored job descriptions
- **Professional Formatting**: Creates descriptions optimized for freelance platforms with proper structure and keywords
- **Instant Results**: Generate complete job descriptions in seconds, not hours

## 🏗️ Architecture

The application follows a modern React architecture with clean separation of concerns:

```
src/
├── components/           # Reusable UI components
│   ├── ui/              # shadcn/ui components
│   ├── Header.tsx       # Navigation header
│   ├── GitHubInput.tsx  # Repository URL input form
│   ├── JobDescriptionResults.tsx  # Results display
│   └── LoadingSpinner.tsx        # Loading state
├── pages/               # Application pages
│   ├── Index.tsx        # Main application page
│   └── NotFound.tsx     # 404 error page
├── hooks/               # Custom React hooks
├── lib/                 # Utility functions
└── main.tsx            # Application entry point
```

### Data Flow

1. **User Input**: User enters GitHub repository URL
2. **Validation**: Client-side validation ensures valid GitHub URLs
3. **API Request**: Application sends repository URL to Lyzr AI agent
4. **AI Processing**: Agent analyzes repository structure, README, and codebase
5. **Result Display**: Generated job description is displayed with formatting options

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ and npm (install with [nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Installation

1. **Clone the repository**
   ```sh
   git clone <YOUR_GIT_URL>
   cd vibecoder-job-generator
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Start development server**
   ```sh
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`
